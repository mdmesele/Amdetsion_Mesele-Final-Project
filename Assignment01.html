Overview of C#

C# is a powerful language that closely follows the traditional C & C++ constructs, but it is more modern and easier to learn.
Developed by Microsoft, this Object-oriented programming language also has a lot in common with Java.
C# code can be compiled on different platforms and comes with a host of strong features such as...

  • Integration with .NET framework
  • Component-oriented
  • High-level structured language
  • Modern syntax; easy to learn
  • Rich standard library
  • Automatic garbage collection
